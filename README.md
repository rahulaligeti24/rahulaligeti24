## 🚀 Featured Projects

### 🤖 [JARVIS – AI Decision Support Assistant](https://github.com/rahulaligeti24/Jarvis)
> An AI-powered personal assistant that helps users capture knowledge and reflect on past decisions using RAG-based knowledge retrieval.

- Built **Retrieval-Augmented Generation (RAG)** for storing and retrieving insights from a personal knowledge base
- Integrated **voice-based interaction** using the Browser Speech Recognition API
- Delivers context-aware AI suggestions to support better decision-making

**Tech Stack:** `Python` `RAG` `MongoDB` `React.js` `Express.js` `Speech Recognition API`

---

### 📝 [InspireHub – Blog Application](https://github.com/rahulaligeti24/blog)
> A full-stack MERN blog platform for creating, editing, and managing blog posts with secure authentication.

- Enforced **JWT-based authentication** and role-based access control
- Integrated **RESTful APIs** with MongoDB for complete CRUD operations
- Built a responsive UI with React.js

**Tech Stack:** `React.js` `Node.js` `Express.js` `MongoDB` `JWT` `bcrypt`

---

- Prevents **scheduling conflicts** via backend validation
